Accessibility Statement

Our Commitment

I am commited to creating and ensuring that this website is accessible toall users. I believe that every user should have a smooth experience browsing my gallery and learning about my work.

Accessibility Features

I have implemented these features for your ease of use.

  • Descriptive alternate text on all images.
  • Color contrast meeting the WCAG AA standards(4.5 ratio for normal text, and 3;1 for large text) with a ratio of 8.59:1.
  • Keybaord navigation that allows all interactive elements to be accessed solely using keybaord.
  • Semantic HTML structure with proper headings and labels.
  • Responsive design that works on mobile devies and larger.
  • Visibile focus indicators for keybaord users.

Testing

This website has been tested using:

  • WAVE Web Accessibility Evaluation Tool
  • Chrome Lighthouse acceddibility audit
  • manual keybaord navigation testing
  • Chrome DevTools responsive desing mode

Website limitations

We are aware of the following accesibility limitations and are working to adress them:

  • Images may slow down the loading time of the website.
  • The contact form is currently not working.

Feedback

If you encounter any Accessibility issues while enjoying the website, please let us know. I want anyone and everyone to have an easy experience I will work to resolve the issues quickly.
